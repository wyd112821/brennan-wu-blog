《我的事业是父亲》读书笔记

# 《我的事业是父亲》读书笔记

# Chapter 1 理念起点篇 养儿育女 特立独行

孩子成为天才还是庸才，不是取决于天赋的多少，而是取决于从出生到五六岁这段时间的教育。

# 拒绝盲从，坚持自我

教子成才的道路上不存在“偏听则暗，兼听则明”的问题，而是“多听则乱，越听越糊涂”。一套大家都满意的家教方法，培养出来的必然是四平八稳的平凡孩子。

有人提倡“走自己的路，让别人去说吧”。我认为人生要想干成一番事业，也只能如此，没有别的两全之策。“祖孙买驴”的故事哲理提示我们，在教子成才的漫长道路上，必须排除干扰，坚定地朝着既定方向前进，走自己的路。如果前面没有路，也要勇敢地闯出一条路来。

# 实施早教，巧抓关键

按照孩子的实际情况，确定一套切实可行的方法进行早期训练。决不轻易听信别人的传言，朝三暮四地改变育儿方法。

在教子成才的道路上，父母的责任就在于替孩子不失时机地抓住每一个关键。

早期教育是一切教育中最重要的教育，是四两拨千斤，牵一发而动一生。

早期教育不仅是智力教育，而且是智力、意志、品德和气概四者合一的教育，是保证人生幸福的教育。

好习惯的培养与好品格的养成是早期教育中非智力因素培养的两个关键，决定了一个人未来发展的高度与人生规模。

# 因材施教，特才特教

早期教育的不同会导致同龄儿童智力发育的不平衡，而智力发育的不平衡必然需要因材施教。

跳级和早读既能为优秀儿童赢得宝贵的时间，又能增加学习的兴趣和积极性。

在基础教育阶段，因材施教的真正含义是对不同的学生用不同速度进行教学。

跳级的具体操作取决于自学能力的培养，只有具备自学能力能够进行超前学习，才谈得上跳级。

家长如果懂得多元智能理论，就不会要求孩子“样样通”，“样样通”的结果必然会导致“样样松”。家长应该发现孩子的长处，发展孩子的优势智能，同时兼顾其他方面的持续发展，使孩子拥有一个健康、健全的人格与智能。

# 智商情商，两翼并举

好的家庭教育，最重要的是把孩子的行为习惯、非智力素质培养好，也就是说要孩子成才必须先教孩子做人，做一个道德高尚、心态良好、人格完备、气概恢宏、意志坚强、自信、自强、自立的人。

道德素质的培养是家庭教育的重中之重，好的道德素质是成才的先决条件。

道德教育的责任不能推给学校，必须由家庭来承担。道德教育不是通过简单的说教就能做好的，而必须由家长多年的言传身教感化出来。

# 超前培养，长远谋划

人生最重要的是能够把目前的努力与未来的希望正确地联系起来。这就需要有超前意识，也就是人们常说的远见。

按照自己的远见来规划孩子的成才策略，既不跟在别人后面跑，也不和别人去争眼前的名次。

# Chapter 2 能力开发篇 一砖一瓦 夯实基础

教育开始得越晚，儿童能力的实现就越少。

# 磨难经历，意外胎教

孕妇的适当紧张可能还会对孩子有益。因为，母亲的紧张会使母体里的胎儿预先做好来到这个困难重重世界的准备。

人类能够给孩子提供的最好教育似乎仍然是苦难。

# 育儿方法，早恒结合

早期的育儿方法应注重细节，特别训练一下孩子几个方面的能力:

一、“专注力”的训练

“注视”训练可以培养婴儿的注意力。这种“专注”习惯的培养，对孩子的注意力集中可能有很大的好处。

二、“胆略”的训练

高举训练一直坚持到他三周岁以后，这对培养他的胆略有很大的好处。

三、不可忽视的“爬行”训练

在家庭的早期教育中，必须设计一个适当的工作量，让孩子完成起来既不轻松也不太难，以能引起其兴趣为度，而且必须以能让孩子完成为止，这样才能得到最好的训练效果。

四、鼓励孩子“成功跨出人生第一步”

跨出这第一步很重要，几次训练下来，他胆子大了，脚步也比较稳了。每次成功之后都给他鼓励，把他高高地向上举起。

五、洗澡和玩水的训练

不管事情多忙，都坚持每天给孩子洗澡，并且让他玩水玩个够。这方法不仅培养了孩子的卫生习惯，而且对孩子的整个身心发育都有很大的好处。

人的各种能力都有各自的发育时间，特别是体能方面也会具有先天的特性，每个孩子的情况可能都不一样，针对不同的孩子，应采取较为个性化的育儿方法。

好的早期教育和训练可以使孩子的好素质得到充分的发挥。

# 语言训练，少中求精

婴幼儿是学习语言的最佳时期，但并不是灌输给他们的语言信息越多越好，学习语言少而精才能入脑，实用性强才能根深蒂固。

背诵诗词和歌谣，除了能对口语和记忆力进行训练，还给日后的气概培养、道德素质熏陶打下了良好的基础。

内容太多并无好处，每天学四句不如每天学两句。学完一遍之后，再重新来一次，复习巩固一下，这不仅有利于记忆，而且会增加孩子学习的兴趣。

三至六岁是儿童识字的最佳年龄，抓住这个最佳年龄段，对培养孩子的智商将起到良好的作用。

背诵过无数相关的诗词，祖国河山的灵光秀气及美好诗词的熏陶，使她在生命的成长过程中得到一种良好的素材，既能陶冶情操，又能塑造灵魂，不仅使孩子一生有良好的胸襟气概，而且会有一颗审美的心灵，去适应变幻莫测的未来，这才是真正的早教啊！

# 思维训练，数学为先

培养孩子的数学能力是开启孩子智力最好的手段之一。

我们坚定地选择数学作为训练孩子思维能力的方法，坚持按既定的早教原则培养孩子。绝不因为轻信和误判而延误了孩子的最佳教育期，使得一个本来可以成才的孩子沦为平庸。

# 能力培养，自学为主

从本质上讲，所有的教育都是在培养学生的学习能力。如果能够有意识地引导孩子独立地进行超前学习，往往会获得意想不到的效果。而自学能力是优秀人才最重要的基本功。

自学能力的培养，不仅为孩子们跳级创造了条件、争得了许多宝贵的黄金岁月，而且更重要的是较强的自学能力会大大促进孩子的学习兴趣和学习能力的开发，为他们在以后的高层次竞争中赢得优势，这种优势越到高处就越明显地发挥出来。

在具体操作上，有如下几点经验：

  1. 家长或老师必须有能力判断孩子是否具备超前学习的能力或潜力。判断的标准有三点：第一是以前的基础是否扎实，第二是对新事物的兴趣，第三是对目标有强烈的追求欲望。
  2. 自学能力必须从小开始培养，以循序渐进的方式进行。
  3. 发现孩子自学潜力不错的时候，要尽快地制定一个阶段性总体目标，让孩子有一个追求的方向，这会使孩子对学习产生强烈的兴趣和动力。
  4. 首先选择自己认为最有优势的课程开始自学。
  5. 自学一门课程时，第一遍要尽量抓紧时间一口气从头至尾学完，目的是获得该课程的整体形象。掌握整体形象之后，第二遍学时就能以整体来理解局部，并能分出重点和非重点，也就会理解得比较深透；第三遍读时再从局部回到整体，使知识系统化，有一个全面的认识。
  6. 在自学过程中遇到什么重大的难关或比较枯燥的基础部分不要怕，能啃多少就啃多少，不能啃的就先搁置下来，把那些内容当成公理接受下来、背下来，大胆地向下冲刺，等学完了下面的东西之后再回头来解决它，可能就很容易了。
  7. 在一个阶段性目标范围内，不必追求100%完全弄懂，重要的是分清主次，把主要的彻底弄懂，把次要的暂时放开。
  8. 自学必须做笔记，对所学内容进行整理，加深理解。
  9. 自学在本质上是培养思考能力，开始时并不在乎很快地学很多知识，重要的是学会如何自学。
  10. 找一位气概恢宏的人作为孩子的导师。导师的作用并不在于由他来给孩子上课、解难题，而是在于他指导孩子如何学习，帮助孩子区分什么是该深入钻研的、什么是可以暂时放过的，什么是重要的、什么是次要的。
  11. 最好有个伴，可以互相比较、相互促进。

# 兴趣培养，注重规范

在培养孩子的兴趣方面，是通过一些巧妙的设计与引导来达到目的。

因为智力教育、道德教育、习惯教育都带有规范性，因此必然需要一定程度的强制性。事实上，教育和学习本质上都是对自然的纠正和对野性的改造，没有强制性也就谈不上教育。

为了培养孩子成才，我们总是咬咬牙，在重大的原则问题上一丝不苟，总是做出一副很严肃的样子，俨然一副严师的外表，该强制的地方，我们从不放松。我始终牢记孟子的名言：“天将降大任于斯人也，必先苦其心志，劳其筋骨……”

# 创新能力，三步养成

培养创造性思维有三个环节：一是学会观察，二是学会阐释，三是学会总结规律，进而能与别人进行交流。

# 艺术教育，点燃灵感

艺术教育是培养孩子自信心的最好手段。美妙的音乐和美丽的画面能使人心情愉快，更容易使孩子产生兴趣。

艺术教育是培养创造性思维的最佳方法。

# 财商培养，经营人生

越早学会理财的孩子，长大后也越会赚钱。

培养孩子延后享受的理念。所谓延后享受，就是指延期满足自己的欲望，以追求自己未来更大的回报。

# Chapter 3 品格养成篇 塑造品格 培养气概

凡人生所需要的重要习惯、倾向、态度，多半可以在六岁以前培养成功。

# 德才兼修，以德为本

道德决定了一个人“志”的方向。志高德高，则孩子成长为优秀人物。

道德教育的责任必须由家庭来承担，有许多方法和技巧：

  • 道德教育不是通过简单的说教就能培养好的，必须由家长多年的言传身教感化出来。
  • 道德教育越早开始越好。
  • 所谓道德教育就是让孩子的言行适应社会约定俗成的规范和国家订立的法则。
  • 道德教育以孝为先。
  • 亲贤人，避小人。

读书成才本身是一条艰苦而漫长的路，没有一种高尚的道德情操和献身精神是不可能读好书的，更不可能成大器。

# 故事熏陶，塑造灵魂

我们每次给他讲故事之后，都要求总结出一个主题，而且反过来要他把故事讲给我们听。这有很大的好处，不仅可以培养他的口语能力，而且还可加深记忆。我们始终认为好的故事不在多，关键是能够引起好的感觉和兴趣，并能深深地烙印在他的幼小心灵之中，成为构筑灵魂的材料。

真正有用的人生哲理其实并不复杂，在几个简单的故事里就能找到做人的原则和应对困境的策略，并且受用一生，传承几代。

# 不断攀登，贵在坚持

如果没有恒心和毅力，即使有伟大的理想，也不可能成就伟大的事业。

在培养孩子的恒心和毅力的具体操作方面，除了自己带头以身作则之外，还要特别注意以下几点：

  • 从小培养孩子从头至尾完成一件事情的能力和习惯。
  • 培养责任感。
  • 有了伟大的目标，你就会把艰苦的劳动视若等闲。
  • 在引导孩子读书成才的过程中，千万不能让孩子有被迫的感觉，要处处培养孩子的主动性。
  • 欲获得强大的毅力，就要多与有毅力的人接近。
  • 让孩子长期坚持某项活动，最好自己带头，这不仅可以锻炼身体，而且能够磨炼意志、培养毅力。
  • 立志和自信是恒心和毅力的前提，胸无大志或缺乏自信的人不可能有强大的恒心和毅力。

# 自信乐观，不断进取

培养孩子的自信得从母爱开始。自幼给予孩子最慈祥、最纯真的爱是培养孩子自信的最根本的环节。

在培养自信的具体操作方面，除了上面所说的那些最根本的环节之外，我们还注意以下几点：

  • 尊重孩子。任何有关孩子的事，不管是生活方面还是学习方面，都要尽可能听从孩子自己的意见，即使我们认为是不正确的，也要尊重他的想法，给出诚恳的建议，从不粗暴地予以否定。
  • 让孩子带路。每次出门，不管是拜访亲友还是外出旅游，总是让孩子走在前头，为我们带路。
  • 除非对号入座，一律挑最前面的位置坐。不论是开会、听课、看演出，都坚持这样做。
  • 经常有意识地与孩子争论问题。自己站在错误的立场上，让孩子在争论中取胜，然后及时予以表扬鼓励。
  • 练习当众发言，培养孩子当众辩论的能力。
  • 经常练习正视别人，用有力的目光注视对方，并且保持笑容。
  • 保持良好的走路姿势，昂首挺胸，两眼看向远方。
  • 经常用暗示的方法，使孩子相信自己是个特殊人物，将来必定有出息。
  • 即使遇到重大的失败，也不能用严厉教训或惩罚的方式对待孩子。相反，应该加以特别的关怀和安慰。
  • 尽量找出优点及时给予鼓励。

用爱去培养和保护孩子的自尊和自信，让孩子一辈子保持着征服者的感情，保持对成功的信心。

# 良好习惯，从小养成

早期的习惯培养对他未来的成才起着决定性的作用。一个没有良好习惯的人往往经不住“魔鬼”的诱惑和“恶兽”的恐吓，结果半途而废，一事无成。

好习惯需要一个很长的过程才能培养出来，而且还要长期自律才能保持，因为好习惯是需要通过自我克制才得以形成的，只有经过不断诱导、启发、磨炼才能慢慢地巩固下来，最后成为一种行为模式；而坏习惯是无须任何努力就能形成的，甚至一次偶然的经历就有可能成为终身恶习。

# 勤俭节约的习惯

通过以下两种方式培养孩子勤俭朴素的习惯：

  • 从孩子们五岁开始，我们便培养他们爱惜财物、懂得节俭的习惯。
  • 让孩子适应艰苦环境中的生活。

# 独立自主的习惯

独立自主的能力和习惯的培养不仅为孩子的生活带来方便，而且使他们能独立地应对人生道路上的各种挑战。

独立的习惯可以派生出其他几个好习惯：

  • 独立的学习习惯。由于有独立自主的习惯，他们能很轻松地掌握自学的方法，独立地进行超前学习，不去依赖老师。
  • 独立的思考能力。独立自主使他们有不轻信传言的习惯，在重大问题上能独立思考，不随波逐流。
  • 创新的能力习惯。独立自主的习惯能提高孩子的思维品质，使孩子们的思维方式富有创意。

# 珍惜时间的习惯

按照计划从头至尾有始有终地完成一件事情，这是一个人的一种能力和修养，但更是一种习惯。这种习惯的培养，对他们长大后做事的能力有很大的好处，做任何事情总是下决心去完成它，从不半途而废，从不轻言放弃。

让孩子学会按计划行事，起码要做好以下两点：

  • 按时定量完成任务。
  • 学会按照计划行事，珍惜时间。

# 远离电视的习惯

  • 长期观看电视是学校教育的头号敌人。
  • 科学研究证明，三岁以下的儿童不宜看电视。
  • 看电视会影响孩子的想象力和人际交往能力的发展。

# 培养自立，始于认路

对孩子辨别道路能力的培养，可以很好地提升孩子各方面的整体能力。

任何一种能力都是训练出来的。人生的路本来就是从不会中走出来的。

# 注重素质，全面培养

在生活方面我们除了培养他朴素、整洁卫生、遵守时间的习惯之外，还特别注意培养他的独立自理能力。

作为素质培养的另一个重要内容是坚持不懈地进行身体锻炼，除了强调参加学校规定的一切体育活动之外，还应每天抽出一定时间跑步、举铁棒，坚持冷水洗面等。

在学习方面，我们强调勤学巧学、博览群书、全面发展，并在此基础上培养兴趣和发展强项。

关于道德素质方面的教育，我们以尊师、诚信为重点。

# 宽容大度，亦忍亦让

一个人的力量在任何时候都是相对微弱的，只有宽容他人，学习他人的优点，才能够弥补自己的不足，实现优势互补，让自己强大起来；也只有学会宽容，才能让自己和他人生活得更加轻松愉快。这种宽容的能力更要从小培养。

须知世界上能成大事业者所必须具备的重要素质是：宽容大度、刚柔并济、亦忍亦让、善进善退。只有这样方能建立人望、取得人心。并且，能忍耐者必能以柔克刚，取得最后的胜利。

# Chapter 4 情境经营篇 教育氛围 主动营造

# 好名相伴，好运相随

好的名字可以为孩子烘衬出良好的成长外围环境，会使人产生自比、自居和自勉效应，对孩子一生的事业肯定有好的促进作用。

# 因时因地，培养孝德

总是利用一切可能的场合向孩子灌输人生的大目标，培养孩子的气概。

在家庭的早期教育中必须将“孝”字刻印在孩子空白的心灵图纸上，使其根深蒂固。否则长大之后，其他一切道德教育都将无从入手。“孝”是对父母之爱的回报，其他一切道德情操是“孝”的衍生。

# 隔代教育，重在陶冶

一个真正读书之家的形成是必定有其根的，没有几代人的共同努力，不可能造就出类拔萃的书香门第。

# 家校合作，教育互补

作为家长，应经常与学校老师沟通，把自己孩子的点滴变化告诉给老师，并希望能与他们进行很好的配合，共同把孩子教育好。

与学校较好配合，使孩子在一种宽松而又正规的氛围里学习，形成良好的心态，自信心和安全感自然而然地得到增强。

# 兄弟互助，相互砥砺

孩子间的相互砥砺所产生的叠加效应对他们的成长都是积极的、共赢的，由此我们可以解释某些独生子女家庭的缺陷，大概就是缺乏这种叠加效应。

兄弟间相互勉励，这种交流是亲子间所不及的。

# 设计旅游，开拓胸襟

旅游可以在孩子幼小心灵的空白图纸上描绘出永远不会褪色的美丽图腾。在未来的岁月中，给他们留下永久美好的记忆，激励他们走向崇高、走向伟大。

我们始终认为伟大的情怀和广阔的胸襟不是通过简单的说教就能培养出来的，而必须经常让美和伟大的外景同崇高的意境凑合在一起才能感化出来。而旅游可以为孩子提供美和伟大的外景，可以增加见识，调节精神，而且也为气概培育提供了外因。

旅行是气概培养的一种绝妙形式，广阔的胸襟和伟大的情怀只能在美的外景和崇高的意境同时具备时才得以形成，所谓感化是不失时机地让两者融合在一起。

# 得之宠爱，失之溺爱

父母对孩子的宠爱是应该的，也是必要的。这不仅不会把她宠坏，而且还会培养她的良好心态，使她永远保持征服者的欲望，保持对成功的信心。所谓宠坏的原因是：你宠他的时候没有去教育他，使他根本就不知道应该怎样做人，不知羞耻、不知恐惧、不知艰难。三不知是没有教育的结果，而不是宠爱的结果。

# 筝线艺术，亲子共勉

与孩子通书信是建立亲子关系的绝妙手段，是其他联络方式所不能比拟的。通过书信与孩子有了心灵通感，才使亲子之间存在一条无形的联络线路，无论彼此离得有多远，这条线总是牢牢地联结着彼此的心。

要想真正能够影响一个孩子的成长，仅靠写几封动人的书信是不容易实现的，还要依赖从小形成的“心灵通感”。

# Chapter 5 人生设计篇 规划成长 建构未来

# 父母爱子，为其远谋

每个孩子身上都蕴藏着巨大的潜能，成功的关键在于我们能否找到发挥他们潜能的目标。在学习上，目标明确，就会激发强烈的学习动机，坚持不懈地为之奋斗。反之，如果没有目标，无所追求，见异思迁，终将一事无成。

# 立志趁早，编织未来

我们所谓的“从小立志”就是从小引导孩子对伟大和高尚的追求和向往，同时从小培养孩子严格要求自己，以历史上和现代的优秀人物为榜样编织自己童年的梦。

立志除了身教感化之外，不失时机地进行说教，诱发出一种崇高的意境，使它与此时此地的美好外景融合在一起，将有一种非常有效的感化效果。

# 专业选择，实用为本

确定个人志向，选好专业，是幸福的源泉、成功的开始。

专业应由本人的兴趣和特长来决定，并结合应用考虑。

# 时间管理，只争朝夕

世界上的一切胜利都可以归结为时间上的胜利。

时间上的超前，无疑要转化为以后竞争上的优势。

小时候的一年比成人后的一年对人生的发展来说更加重要，时间管理的重要性也在这里。

# 抓住机遇，把握命运

人生的关键处往往就差那么一两步，孩子还小，没有能力去把握自己的命运，父母的责任就在于不失时机地抓住每一个有利于成才的关键点，不放过任何一个机遇。

远见和超前意识帮助我们区分什么是孩子成才真正重要的关键，又能不失时机地抓住每一个关键。

# 不断攀登，永不满足

不断地为孩子们设定并调整更高追求的目标，不安于现状，不在胜利面前自我陶醉。这是走向更大成功的关键所在。

# 焦点问题经验分享

# 为什么把教子作为自己的事业去追求？

一些父母把养育儿女当成艰苦的义务，并常常暗示甚或明示子女要孝顺、回报，长大后要为父母做什么，等等。这就像是对待我们自己的工作，我们都知道如果只把工作看成是自己的职责、自己谋生的职业，则往往重视的是有付出就要有回报，做了多少就要给我多少酬劳。

而事业则不然，在追求事业成功的过程中，重视的就不仅仅是回报，更多的应该是在过程中享受到乐趣。如果我们把做父母看作我们终身的事业，那我们抚养、教育孩子就是一种命运赐予的享受：享受和子女相处的时光，享受看着他们成长的乐趣。所以今天我想强调的就是做父母是我们终身的事业。那样，我们的家庭教育就应该是一种享受。

教子不仅仅是学校的事情，更重要的是父母的事情。

# 那套在外人看来近乎“魔鬼”的家教模式下，孩子会快乐吗？

我们的孩子从来都没有在高压下被动地学习。相反，他们是在轻松、愉快的环境下学习的，也正是由于这种愉悦的学习体验保障了他们持续旺盛的学习动力和学习兴趣，实现了一种自我激励式的自觉学习。

丰富的文体活动在孩子们的童年里积淀了多彩多姿的美好回忆，使得他们体会到了人生的乐趣，从而更容易懂得如何去追求波澜壮阔的人生规模，实现真正意义上的美丽人生。

# 孩子青春期逆反、不服从大人的管理，怎么办？

现在的父母聚在一起，总说自己的孩子难管、不听话。殊不知，孩子不听话是家长一手造成的。

作为父母，应经常问问自己要求孩子做的事情自己是否做到了。如果父母光说不做，对孩子的说教将是多么苍白！当然很多时候不是我们做父母的不明白以身作则的道理，只是没有意识到这种以身作则应该是随时随地、持之以恒的，认为偶尔的行为对孩子不会有什么影响，这样的想法是非常不可取的。可能正是由于父母不可取的“偶尔”“点滴”日积月累而造成了孩子的不听话。

孩子年小幼稚，如同一张白纸，对父母的言行深信不疑。父母正确的言行会使孩子学到正确的处世方式；反之则会使孩子是非不分，甚至形成不良的行为习惯。一方面，孩子会非常注意父母的言行，确定他们的言与行是否一致；一方面则加以模仿，并在不断的模仿中形成道德标准。

所以，家长对孩子的教育不要再步入一个误区，认为对孩子的教育就是给孩子讲道理，进行说教。殊不知身教胜于言教，要孩子做到的事必须自己先做到，用斗志去激发斗志，用气概去培养气概，用道德去感化道德，用对伟大的追求去引导孩子走向伟大。

# 为什么不要不张扬孩子的成绩，不要给孩子预支鲜花。

父母发现孩子很优秀，拼命地给孩子到处宣扬、预支鲜花，只会徒然增加孩子的心理压力。我们所有的孩子考上大学、出国留学都未曾专门在家摆过酒席，我们尽量让孩子不露锋芒，保持平静，使他们在没有压力的环境中，轻松地按照自己的愿望去发展自己，而不必不断地向别人证明自己的优秀，这大大有利于他们的成长。

还有，很多家长总喜欢把优秀的孩子当作自己孩子的参照，孩子一旦表现不合自己的意愿，父母就会埋怨。这种指责和比较除了加重孩子的心理负担外，不能解决任何问题。把自己的孩子和别人的孩子相比，只能给自己带来不满和痛苦，又伤害孩子的自尊心。其实孩子自身的超越和进步才是最值得家长关注的。在学习上，取得进步就是优秀，对孩子的要求是每天进步一点点。

家长要多花一些精力去真正关注孩子的内心体验，注意到他的每一次努力和每一个小小的进步，不给他设立过高的目标，信任孩子，信任比惩罚更能够激起责任心；永远做孩子前进路上的欣赏者，持之以恒地给予孩子欣赏和鼓励。如果上述这些家长都能做到，孩子真的就会发生神奇的变化。

    插件类

    # 插件类

    # ECharts

    ECharts (opens new window),一个基于 JavaScript 的开源可视化图表库。

    # Highcharts

    Highcharts (opens new window),兼容 IE6+、完美支持移动端、图表类型丰富的 HTML5 交互图表库。

    # AntV | 蚂蚁数据可视化

    AntV (opens new window) 是蚂蚁集团全新一代数据可视化解决方案，致力于提供一套简单方便、专业可靠、无限可能的数据可视化最佳实践。

    # vue-echarts

    vue-echarts (opens new window) Apache ECharts 的 Vue.js 组件，同时支持 Vue.js 2/3。

    # v-charts

    v-charts (opens new window) 基于 Vue2.x 的 Echarts 组件

    # Velocity.js

    Velocity.js (opens new window):一个非常快的动画引擎。

    # Three.js

    Three.js (opens new window),是一款开源的主流3D绘图JS引擎。

    # G6

    G6 (opens new window),是一个简单、易用、完备的图可视化引擎。

    # Swiper中文网

    Swiper(Swiper master) (opens new window)是目前应用较广泛的移动端网页触摸内容滑动js插件。

    # jQuery插件库

    jQuery插件库 (opens new window),收集最全最新最好的jQuery插件

    # jQuery之家

    jQuery之家 (opens new window),自由分享jQuery、html5、css3的插件库

    # Vue 插件库

    Vue 插件库 (opens new window),Vue插件,Vue 中文插件大全,Vue 插件使用教程,Vue 插件下载

    # VeeValidate

    VeeValidate (opens new window),Vue.js的表单验证

    # Moment.js

    Moment.js (opens new window) 是一个解析，验证，操作和显示日期和时间的 JavaScript 类库。

    # Day.js

    Day.js (opens new window) 是 Moment.js 的 2kB 轻量化方案，拥有同样强大的 API。

    # wangEditor

    wangEditor (opens new window) 是 Typescript 开发的 Web 富文本编辑器， 轻量、简洁、易用、开源免费。

    # TinyMCE

    TinyMCE (opens new window) 是一款易用、且功能强大的所见即所得的富文本编辑器。

    # js-cookie

    js-cookie (opens new window) 一个用于处理cookie的简单、轻量级JavaScript API

    # ckplayer

    ckplayer (opens new window) 支持http协议下的flv,f4v,mp4,支持rtmp视频流和rtmp视频回放，支持m3u8格式，是你做视频直播，视频点播的理想播放器。

    # 西瓜播放器

    西瓜播放器 (opens new window) 一款带解析器、节省流量的HTML5视频播放器。

    # DPlayer

    西瓜播放器 (opens new window) 一款可爱的HTML5 danmaku视频播放器。
